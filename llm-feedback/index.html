LLM ASCII Art Evolution

How it Works

This example demonstrates a feedback loop with an AI language model to create evolving ASCII art. Here's what's happening:

  1. The AI is given a system prompt that defines its role as an ASCII art generator.
  2. When you click Start, it generates an initial piece of ASCII art.
  3. For each iteration, the previous art is fed back to the AI with a request to evolve it.
  4. The temperature setting controls how creative/random the AI's responses are.
  5. The delay controls how frequently new art is generated.

System Prompt:

You are an ASCII art generator. Create abstract evolving patterns. Only respond with ASCII art, no explanations.

Initial User Prompt:

Create a simple ASCII art scene. Keep it under 20 lines and 40 characters wide. Make it abstract and evolving.

Evolution Prompt Template:

Evolve this ASCII art into something different while maintaining some connection to the previous version.
